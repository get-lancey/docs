### Prerequisites

- Administrator level access to generate API keys within Zendesk

<Note> Lancey will bring in last 30 days of tickets by default. If you'd like to change this limit, mention this to your Lancey point of contact. </Note>

### Zendesk Instructions
<Steps>
  <Step title="Admin Center">
    Navitate to **Admin Center** by clicking on the grid icon on the top right of the Zendesk menu.

  <img src="/images/zendesk-admin.png" />

    
  </Step>
  <Step title="Apps and Integrations">
    Under the left sidebar find the **Apps and Integration** section and click on the **Zendesk API** link.
  </Step>
  <Step title="Enable Token Access">
    Make sure Token Access is enabled in order to generate API token.
  </Step>
  <Step title="API Token">
    Click on **Add API Token**. Your new API token will be displayed. Make sure to copy this token immediately and store it securely, as you won't be able to see it again. 
  </Step>
</Steps>

Now that you have your API details from Zendesk, it's time to head over to your [Lancey dashboard](https://app.lancey.ai). 

### Lancey Instructions
<Steps>
  <Step title="Finding Zendesk integration">
    Use this [link](https://app.lancey.ai/settings/integrations/zendesk.com/manage) to navigate to the Zendesk integration inside of Lancey.
  </Step>
  <Step title="API Url">
    Paste in your Zendesk subdomain specific URL into this field. 
    
    This is the sample format, make sure to replace the subdomain portion with your company specific version:

    https://replacesubdomain.zendesk.com
  </Step>
  <Step title="Email">
    Type the email of the account that generated the API token. 
  </Step>
  <Step title="API Key">
    Paste in **API Token** from Zendesk you saved into this field. 
  </Step>
  <Step title="Test connection">
    Once you've provided the three required fields, press the **Connect using credentials** button
  </Step>
</Steps>
